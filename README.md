# File Metadata Extractor

A responsive web application that extracts metadata from uploaded Images and PDF files using the Browser File API and EXIF.js.

## Features

- Upload Images
- Upload PDFs
- Drag & Drop Upload
- Image Preview
- PDF Preview
- File Name
- File Size
- MIME Type
- Last Modified Date
- Image Dimensions
- Camera Make
- Camera Model
- Lens Model
- Date Taken
- GPS Coordinates
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser File API
- EXIF.js
- Font Awesome

project/

## How to Run

1. Download or clone the repository.

2. Open the project folder.

3. Open `index.html` in a browser.

For best results, use the Live Server extension in Visual Studio Code.
